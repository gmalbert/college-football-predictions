# Copilot Instructions for College Football Predictions

This repository is a Streamlit app for college football predictions and betting analytics.

## Primary goals

- Implement a data-driven prediction and betting dashboard using CFBD + ESPN APIs.
- Keep production code clean and readable.
- Prioritize modern Streamlit patterns and compatibility.

## Style & constraints

- **Do not use** `use_container_width=True` anywhere in the app; this option is deprecated.
- Prefer `width="stretch"` for responsive full-width components and `width="content"` for fixed-width widgets.
- Prefer explicit responsive layouts using `st.columns`, `st.expander`, and CSS-friendly layout blocks.
- Cache expensive data loads and model inference with `@st.cache_data` / `@st.cache_resource`.
- Avoid stateful side effects on page reruns; keep functions pure when possible.

## What is in this repo

- `predictions.py` as the main entry point
- `utils/` library wrappers for CFBD and ESPN APIs, storage, features, and model utilities
- `docs/` with planning and roadmap docs for data, models, features, layout, deployment, pipeline, and betting strategy.
- `.streamlit/config.toml` for local settings
- `requirements.txt` for dependency pinning

## UX rules

- Always include a responsible gambling disclaimer.
- Use consistent team branding from `data_files/logo.png`.
- Keep the side panel as a navigation/help/metrics area.

## Deployment

- Streamlit Community Cloud
- Use secrets (no API keys committed)
- Scheduled refresh through GitHub Actions
